HTTP

When importing HttpClientModule you may get the following error:

Error: Fetch error: 404 Not Found
Instantiating http://localhost:3000/tslib
Loading http://localhost:3000/node_modules/@angular/common/bundles/common-http.umd.js

You need to add BOTH the following to your systemjs.config.js file.

'tslib': 'npm:tslib/tslib.js',
'@angular/common/http': 'npm:@angular/common/bundles/common-http.umd.js',
